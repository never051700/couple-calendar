# 저작자 표시와 라이선스

이 저장소는 친구가 처음 만든 Android용 프로젝트를 iPhone과 Android가 함께 쓰는
공용 프로젝트로 보강해 인계하기 위해 공개한 사본입니다.

현재 원저작자의 이름과 GitHub 계정은 공개 동의를 받기 전이라 문서에 기재하지
않았습니다. 저장소 관리자는 친구와 다음 항목을 확인한 뒤 이 문서를 갱신해야
합니다.

- 최초 작성자와 이후 기여자의 공개 표시 방식
- 코드와 이미지 자산의 소유권
- 외부 사용·복제·수정을 허용할 범위
- 적용할 오픈소스 라이선스

현재 별도의 `LICENSE` 파일이 없으므로, 이 저장소가 공개되어 있다는 사실만으로
제3자에게 복제·수정·재배포 권한이 부여되지는 않습니다.
